This folder contains R script and data files to reproduce the functional data analysis on EEG signals on the paper:

"Stronger Central Mu Suppression for Neutral Than Disgusted Faces During Emotion Categorisation"
by 
Özge Karakale, Doan Do, Ana Arribas-Gil, Fernando Marmolejo-Ramos
(2026)

Contents:

- CSV_files: folder including all csv files containing the CWTs for each task/condition/participant/electrode/band for both baseline and post-stimulus periods.

- Descriptive_Graphics_4electrodes.R: reproduces Figures 4 and 6 in the paper

- EEG_CWT_ratios_differences_GlobalTesting.R: implements all functional confidence bands analysis and reproduces Figures 2, 3, 5, S1-S4 in the paper
